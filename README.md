# Beach World Cup Qualifier Sim

## Current Version: V2

**IMPORTANT**: Make sure the _.exe_ and _settings.json_ files are in the same folder.

======================

## _Settings_

**simSpeed** - How fast the sim will play. Set to 0 to be instant.


**matchMultiplier** - How much of an advantage does the nation with higher points get over the lower nation.


**displayChanges** - Display at the end of the simulation the _boosted/nerfed_ nations.


**topTeamPlacement** - How high up in rankings does the nation need to be to be considered top-tier


**topTeamCanBoost** - Allows for top-tier nations to still receive boosts in simulations.


**maxBoosts/maxNerfs** - Max amount of nations for each. Always random from 1 to whatever the max is set.


**maxBoostPercent/maxNerfPercent** - The highest % a nation can be boosted/nerfed.


**europe/africa/asia/north america/oceania/south america** - Qualification spots. Setting a nation as host will remove a spot from their respective region.
